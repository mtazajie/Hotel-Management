# Hotel Management System (C++)

This is a simple console-based Hotel Management System written in C++.

## Features
- Add and manage room records
- Store customer and booking information
- Save and retrieve data using file handling
- Menu-based user interaction

## What I learned
- C++ programming basics
- File handling using `.dat` files
- Program structure and control flow
- Logical problem-solving

## Technologies
- Language: C++
- Environment: Console application

## Purpose
This project was built as a learning exercise to strengthen my understanding of programming fundamentals and system logic while preparing for engineering studies.
